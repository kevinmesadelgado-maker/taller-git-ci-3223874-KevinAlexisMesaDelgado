# taller-git-ci-3223874-kevinAlexisMesaDelgado

Proyecto del taller de Git y CI (semanas 1 a 6).  
Incluye una interfaz simple con una lista y un botón, y funciones básicas en JavaScript.

## Funcionalidades

- Lista en pantalla (`#lista`).
- Botón **Agregar** para añadir elementos.
- Función `sumar(a, b)`.
- Función `agregarItem(texto)` que crea un `<li>`.

## Archivos principales

- `scr/index.html`
- `scr/app.js`
